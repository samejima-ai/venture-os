## 判定原則（D3）

- **IT-active**: 価値論理がデジタル経済性（複製限界費用ゼロ・在庫レス・地理制約レス）で成立 → L0 選択対象
- **physical-frozen**: 物理ロジ・実在庫・逆物流・実店舗立地が価値の核でデジタル再来なし → 参照凍結（思想温存・選択非対象）
- **dual-face**: 同一抽象で物理面を凍結・デジタル再来面を active（二面管理）

> **正典カバレッジ注記**: 本データは正典（拡張編）に記載の27＋5＋第III部パターン、および正典が相互参照する
> 基幹35パターンの被参照分を分類する。**基幹35の完全な逐語分類は基幹辞典本体（本 repo 未収録）取得後に確定**する。
> 現時点で名称のみ推定の基幹パターンは `〔基幹・要辞典本体〕` を付す。

## G 市場ポスチャ軸 → IT 類型（D4）

BM パターン（何で稼ぐか）と直交する第一級軸。大辞典 G の思想を IT ポスチャに翻訳（考案 v2 §4）。

| G パターン（思想） | IT 市場ポスチャ類型（軸の値） |
|---|---|
| No Frills（G-2） | セルフサーブ/PLG 最小機能・下層独占（開発者/個人） |
| Ultimate Luxury（G-4） | エンタープライズ専任・高ACV・少数アカウント |
| Aikido（G-5） | アンバンドリング/逆張り単機能特化 |
| Target the Poor（G-1） | 従量マイクロ課金・未開拓セグメント単位経済再設計 |
| Reverse Innovation（G-3） | 制約市場（低帯域/モバイル/規制産業）起点→先進市場還流 |
| （既定値） | mainstream horizontal / vertical SaaS |
